> Markdown 是一种轻量级的标记语言，广泛用于在各种平台上写作、格式化文本和创建内容。它易于阅读和书写，同时也可以转换为 HTML 等格式。

## 标题

使用 `#` 符号表示标题级别，一个 `#` 表示最高级别的标题，依此类推，最多支持六级标题。

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

## 文本样式

- **加粗文本**：用两个 `**` 包围文本。
- *斜体文本*：用一个 `*` 包围文本。
- ~~删除线文本~~：用两个 `~~` 包围文本。
- ==文本高亮==：用两个 `==` 包围文本。
- <u>文本下划线</u>：用 `<u> </u>` 包围文本。

```
**加粗文本**
*斜体文本*
~~删除线文本~~
==文本高亮==
<u>文本下划线</u>
```

## 列表

无序列表使用 `-` 开头。

```
- 无序列表项1
- 无序列表项2
  - 嵌套的无序列表项
  - 嵌套的无序列表项
```

- 无序列表项1

- 无序列表项2
    - 嵌套的无序列表项

    - 嵌套的无序列表项


有序列表使用数字后加点和空格。

```
1. 有序列表项1
2. 有序列表项2
3. 有序列表项3
```

1. 有序列表项1
2. 有序列表项2
3. 有序列表项3

## 引用

使用 `>` 符号表示引用文本。

```
> 这是引用的文本。
> 引用可以跨行。
```

> 这是引用的文本。
> 引用可以跨行。

## 表格

创建表格时，使用 `|` 符号分隔不同的列。

```
| 列1    | 列2   | 列3     |
| ------ | ----- | ------- |
| 内容1  | 内容2 | 内容3   |
| 内容4  | 内容5 | 内容6   |
```

| 列1   | 列2   | 列3   |
| ----- | ----- | ----- |
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
